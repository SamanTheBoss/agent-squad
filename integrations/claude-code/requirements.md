# /agent-requirements

Run the requirements agent to interview the user
and produce spec files for the project.

## Usage
/agent-requirements

## What this does
1. Interviews you about what you are building
2. Asks one question at a time
3. Detects conflicts and protects MVP scope
4. Produces all spec files in context/

## Output files
- context/project.md
- context/features.md
- context/design.md
- context/technical.md
- context/test-criteria.md
- context/roadmap.md
- context/open-questions.md
